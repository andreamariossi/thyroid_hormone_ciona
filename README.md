# Code and analysis outputs

Supporting materials for the manuscript: Mariossi & Levine **_A Thyroid Hormone-Mediated Opsin Switch Initiates Metamorphosis in a Proto-Vertebrate_**  

This repository contains R notebooks/scripts, source (quantification) data, and rendered figure reports used to reproduce the analyses, plots, and statistics reported in the paper. (to fill up later at publication)

---

## Repository contents

### Folders
- `code/` — R scripts used to process data and generate plots
- `figure/` — exported figure panels (main and supplementary)
- `data/` — source/quantification files used in the analyses

### Rendered analysis reports (HTML)
These HTML files are self-contained figure reports and can be opened in any browser:
- `Fig-1A_thyroid receptor.html`
- `Fig-2BC_tail regression- T4 & T3.html`
- `Fig-2C_tail regression- T4 window.html`
- `Fig-S5H_thr_qPCR_nkx2.1-KD.html`
- `Fig-S6_S7_endodermal strand.html`
- `Fig-S9_T4 & T3 RTqPCR.html`

> If GitHub does not render an HTML file correctly, download it and open it locally (Chrome/Safari/Firefox).
